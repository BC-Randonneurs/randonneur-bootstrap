## Requirements

- `yarn`

How to customize:

# `yarn` to install

# Make your edits

# `yarn build-project`

If you have your directory structure as follows:

```
your-rando-root/
├── bcrdata/
│   └── bcrdatabase/
└── randonneur-bootstrap/
|
...
└── randonneur-wp/
```

You can run:

`yarn build-db`

to copy the files to the right directory
